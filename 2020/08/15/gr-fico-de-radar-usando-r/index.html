Gráfico de radar usando R

2020-08-15

El gráfico de radar es una representación visual que se puede usar para describir el nivel de habilidades de diferentes jugadores. Los videojuegos como FIFA o PES suelen usarlos, para describir y presentar la habilidad de cada jugador. En este artículo se presenta el proceso de creación de estos gráficos usando el lenguaje de programación R.
